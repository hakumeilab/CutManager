主な機能

取り込み

素材名から拾えます

素材フォルダー名や動画名から、カット情報を自動で整理できます。

整理

ソートやフィルターで整理できます

納品前の確認や進行に合わせて、必要なカットを絞り込みながら見やすく整えられます。

軽快

軽く編集できます

CSVベースなので、Excelを開くより軽く管理できます。

更新

更新もすぐ確認できます

ボタン一つで最新版を確認できるので、配布や更新の手間を抑えやすくなります。
